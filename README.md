# how to setup:



### 1. download the latest release of BeatSaberFix
### 2. unzip TheaterFix.zip
### 3. move/copy TheaterFix\Libs\Theater\yt-dlp.exe to yourbeatsaberinstall\Libs\Theater\yt-dlp.exe (there is an already existing yt-dlp.exe in there so replace it with the new one)
### 4. set the new yt-dlp.exe to Read Only (right click yt-dlp.exe -> properties -> check Read-only -> apply)
### 5. in yourbeatsaberinstall\Libs\ create a new folder called "TheaterFix"
### 6. move/copy TheaterFix\Libs\TheaterFix\yt-dlp-real.exe to yourbeatsaberinstall\Libs\TheaterFix\yt-dlp-real.exe
### 7. add this chrome extension to google chrome "https://chromewebstore.google.com/detail/get-cookiestxt-locally/cclelndahbckbenkjhflpdbgdldlbecc"
### 8. go to youtube.com and log in with your account
### 9. click on the extensions icon (found near the top right corner in google chrome and it looks like a puzzle peice)
### 10. in the dropdown menu click "Get cookies.txt LOCALLY"
### 11. click on export
### 12. rename the downloaded file to cookies.txt
### 13. move cookies.txt to yourbeatsaberinstall\Libs\TheaterFix
